
## Mini-Marcket
## Demo

Crear una Mini tienda básica donde:

El usuario pueda ver una lista de productos (nombre, precio, categoría, descuento).

Pueda agregar productos al carrito.

El sistema calcule el subtotal, descuento total, impuesto y total final.

El usuario elija un método de pago (tarjeta, PayPal, cripto).

Se procese el pago y se muestre una transacción final con los detalles.
